# 班级成长探险岛 🏝️

超萌游戏化班级管理工具 - 纯前端H5版本

## ✨ 功能特点

- 🎯 **只奖励不扣分**：全正向激励，让孩子主动争着表现
- 🐾 **萌宠进化系统**：从蛋壳宝宝到海岛守护者的趣味成长路线
- 🏝️ **班级海岛共建**：全班星星总和推动海岛建设进度
- 💝 **AI软萌评语**：自动生成温柔可爱的鼓励评语
- ⚡ **一键批量加星**：快速为全班或单个幼儿添加星星
- 🏆 **光荣榜展示**：只展示进步，不公示落后

## 🚀 快速开始

### 方式一：直接打开
双击 `index.html` 文件即可在浏览器中运行

### 方式二：部署到 Vercel（推荐）

1. 将项目上传到 GitHub
2. 访问 [vercel.com](https://vercel.com)
3. 点击 "New Project"
4. 导入你的 GitHub 仓库
5. 点击 "Deploy"
6. 获得免费域名，如：`your-project.vercel.app`

### 方式三：部署到阿里云 OSS

1. 登录阿里云控制台
2. 创建 OSS Bucket，设置为静态网站托管
3. 上传 `index.html` 文件
4. 获取访问域名

## 📱 使用说明

### 首次使用
1. 点击"进入演示模式"或"教师登录"
2. 添加幼儿姓名（可单个添加或批量导入）
3. 开始使用各项功能

### 核心功能

#### 1. 每日探险日报（首页）
- 查看今日探险口号
- 查看今日3个小探险任务
- 查看班级海岛建设进度
- 获取1分钟课堂小游戏灵感
- 一键复制家园共育短句

#### 2. 萌宠队员列表
- 查看所有幼儿的萌宠形象、星星数、等级
- 点击幼儿卡片可进行加星操作
- 支持添加新队员和批量导入

#### 3. 快速打卡加星
- 全班一键加星（每人+1）
- 按行为标签批量加星
- 单独点击幼儿加星

#### 4. AI软萌鼓励评语
- 选择幼儿和行为标签
- 自动生成温馨评语
- 一键复制发家长群

#### 5. 班级星光看板
- 今日全班星星总数
- 本周星光榜（前5名）
- 最新进化播报
- 小组探险队PK对比

#### 6. 设置与数据
- 导出数据为CSV格式
- 升级付费版
- 清空所有数据
